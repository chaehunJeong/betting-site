# 랜덤 이미지 선택 사이트 개발 프로젝트 TODO LIST

## 📋 프로젝트 개요
직장인들이 카드나 물건 사진을 찍어 올리면 그 중 하나를 랜덤으로 선택해주는 웹사이트 개발 및 애드센스 수익화

**사용 예시**:
- 점심 메뉴 5개 사진 찍어 올리기 → 오늘 먹을 메뉴 랜덤 선택
- 커피 브랜드 카드 올리기 → 누가 살지 랜덤 선택
- 게임 아이템/캐릭터 선택
- 여행지, 영화 등 결정 장애 해결

## 📝 메모 및 중요 팁

### 개발 관련
- 초기에는 간단하게 시작하고 사용자 피드백을 받아 개선
- **백엔드 없이** 프론트엔드만으로 구현 가능 (완전 무료)
- 모바일 우선 설계 (대부분 사용자가 스마트폰으로 사진 촬영)
- 이미지는 클라이언트 메모리에만 저장 (개인정보 보호)

### Cloudflare Pages 장점
- **완전 무료**: 무제한 대역폭, 무제한 요청
- **빠른 속도**: 전 세계 CDN 네트워크
- **자동 배포**: Git push만 하면 자동으로 배포
- **미리보기**: PR마다 자동으로 미리보기 URL 생성
- **환경 변수**: 무료로 환경 변수 설정 가능

### 애드센스 수익화
- 애드센스 승인은 사이트 오픈 후 신청 (콘텐츠와 트래픽 필요)
- 최소 10-20개의 페이지/콘텐츠 필요 (정책 페이지 포함)
- 광고는 사용자 경험을 방해하지 않는 위치에 배치
- 첫 달 수익은 미미할 수 있으니 장기적 관점 필요

### 차별화 포인트
- **속도**: 이미지 업로드 → 선택까지 10초 이내
- **간편함**: 회원가입 없이 바로 사용
- **재미**: 화려한 애니메이션 효과
- **공유**: SNS로 결과 공유 (바이럴 효과)

### 확장 가능성
- 게임 아이템 뽑기
- 음식점 추천 (근처 식당 사진)
- 선물 추천
- 팀 구성원 랜덤 선택
- 복권 번호 생성 (숫자 이미지)
